### 증감연산자

| 증감 연산자의 종류 | 증감연산자의 의미                |
| ---------- | ------------------------ |
| ++a        | a의 값을 1 증가시킨 후 연산을 진행한다. |
| a++        | 연산을 진행한 후 a의 값을 1 증가시킨다. |
| --a        | a의 값을 1 감소시킨 후 연산을 진행한다. |
| a--        | 연산을 진행한 후 a의 값을 1 감소시킨다. |

### 비트 연산자

| 비트 연산자의 종류 | 비트 연산자의 의미                  |
| ---------- | --------------------------- |
| ~          | 비트를 반전시킨다. (NOT)            |
| ^          | 두 개의 비트가 달라야 1이다. (XOR)     |
| &          | 대응되는 비트가 모두 1일 때 1이다. (AND) |
|            |                             |
| <<         | 지정한 수만큼 왼쪽으로 비트를 이동시킨다.     |
| >>         | 지정한 수만큼 오쪽으로 비트를 이동시킨다.     |
